About SensiMAR

SensiMAR is the given name to a multisensory project created in the scope of a PhD in Informatics, in University of Trás-os-Montes and Alto Douro, Vila Real (Portugal). Sensi was inspired in the word «senses» and MAR stands for «Multisensory Augmented Reality».

This project includes the following tasks:

  • Literature Review
  • Technology Acceptance Study
  • Multisensorial AR Prototype
  • Collection Results
  • Discussion of Results

A work plan is presented to describe the tasks and milestones intended for this research project:

Literature Review

This task includes the following chores: Theoretical Foundations and Previous Studies Analysis. The milestones stablished for this task are creating the Research Questions, Developing a Systematic Review, and present a State of The Art regarding to Multisensory applications in cultural heritage contexts.

Technology Acceptance Study

This task includes the following chores: Literature Review of the existent models, Creating questionnaires, and Questionnaires delivery. The milestones stablished for this task are the Questionnaires (PT and EN versions), and a Acceptance and Intention to Use Technology proposal model.

Multisensorial AR Prototype

This task includes the following chores: Requirements Gathering, Developing the Prototype, Audiovisual component: Pilot-tests, Olfacotory compontent: Pilot-tests, and Implementation in-situ. The milestone stablished for this task is a functional Multisensorial AR system.

Collection Results

This task includes the following chores: Creating the questionnaires and Collecting Results in-situ. The milestone stablished for this task are Users’ answers.

Discussion of Results

This task includes the following chores: Analysing the data and Discussion of Results. The milestone stablished for this task is a Hypothesis Validation.
